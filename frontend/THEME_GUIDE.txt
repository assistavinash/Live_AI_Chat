/* SIMPLE THEME SYSTEM GUIDE */

THEME COLORS:

Light Mode (Default):
- Background: White
- Text: Dark Gray/Black
- Button: Blue (#007bff)
- Border: Light Gray

Dark Mode:
- Background: Dark Gray (#1a1a1a)
- Text: White
- Button: Dark Blue (#0056b3)
- Border: Medium Gray

HOW TO USE:

1. Theme Switcher Button
   - Located in top-right corner
   - Click to toggle between Light and Dark modes
   - Preference is saved to localStorage

2. CSS Variables in theme.css
   - Use any color variable in your CSS
   - Example: color: var(--text-primary);
   - Variables automatically change with theme

3. Files:
   - src/styles/theme.css - CSS variables only
   - src/styles/auth.css - Mobile-first responsive styles
   - src/components/ThemeToggle.jsx - Theme switcher component
   - src/components/ThemeToggle.css - Button styling

No gradients, no complex animations - just clean and simple!
